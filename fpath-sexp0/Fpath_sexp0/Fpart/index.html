Module Fpath_sexp0.Fpart

Part of a file path.

A Fpart.t represents a segment of a file path, i.e., the parts of the path that are separated by the directory separator character.

For example, in the file path "/home/user/documents/file.txt", the parts are ["home" ; "user" ; "documents" ; "file.txt"].

A valid file part cannot contain '/' or null characters.

This module provides functions to convert between strings and file parts, validate file parts, and some common file parts.
